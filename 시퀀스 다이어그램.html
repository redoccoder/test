  alt 인증 성공
    AppServer ->> Client: 로그인 성공 응답
    Client ->> User: 로그인 성공 표시 및 계정 접속
    User ->> Client: 송금/결제 메뉴 선택
    Client ->> AppServer: 거래 정보 전송 (수취인, 금액 등)
    AppServer ->> DB: 거래 정보 임시 저장
    AppServer ->> Client: 거래 내용 확인 요청
    Client ->> User: 거래 내용 확인 표시
    User ->> Client: 최종 확인 (예/아니오)
    Client ->> AppServer: 최종 확인 응답
    alt 최종 확인 예
      AppServer ->> PaymentGateway: 결제 승인 요청
      PaymentGateway ->> Bank: 자금 이체 요청
      Bank -->> PaymentGateway: 자금 이체 결과 응답
      PaymentGateway -->> AppServer: 결제 승인 결과 응답
      alt 결제 성공
        AppServer ->> DB: 거래 결과 저장 (성공)
        AppServer ->> Client: 거래 성공 응답
        Client ->> User: 결제 완료 표시 (영수증 등)
      else 결제 실패
        AppServer ->> DB: 거래 결과 저장 (실패)
        AppServer ->> Client: 거래 실패 응답 (오류 메시지)
        Client ->> User: 결제 실패 표시
  AuthServer -->> AppServer: 인증 결과 응답
      end
      AppServer ->> Client: 사용자에게 알림 전송 (성공/실패)
      Client ->> User: 알림 표시
    else 최종 확인 아니오
      AppServer ->> DB: 임시 거래 정보 삭제
      AppServer ->> Client: 거래 취소 응답
      Client ->> User: 거래 취소 표시
    end
  else 인증 실패
    AppServer ->> Client: 로그인 실패 응답 (오류 메시지)
    Client ->> User: 로그인 실패 표시
  end
  Client ->> User: 세션 종료 또는 추가 작업 대기
